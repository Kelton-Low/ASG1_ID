# Twice Website

This project is a website for the K-Pop girl group TWICE, inspired by the groupâ€™s aesthetic and recent styles.
The site includes a homepage, individual member pages, and a discography section.

This website would be for their fanbase ONCE, who come here to check out the different songs released by TWICE and find more information about their favourite members



## Features
1. Sound player to play the song
    - In the main page, users can play the latest album by TWICE
2. Carousel to display different members of TWICE
    - In the profile page, users can scroll through the different members within the group by clicking on the arrows
3. Showcase of the discography of TWICE
    - Users can see in the discography page the many albums that TWICE has released, and sort them by date or alphabetical order
4. Menu to navigate between the different pages
    - Users can click a menu icon in the top right hand corner to navigate between pages
5. Links to different social media pages and different albums of TWICE
    - Users can click on various icons and headers in the website to be redirected to the different websites and socials
